# 聚會報名系統

這是一個為聚會報名與出席統計設計的輕量系統，讓同工可以建立聚會、開放報名、統計參加人數與餐點需求，並在後台集中管理報名資料。

系統以手機使用情境為主，適合透過 LINE 或行動瀏覽器分享報名連結。

## 主要功能

- 開放聚會報名
- 可查詢既有報名並更新資料
- 管理者以帳號密碼登入後查看報名名單
- 可透過初始設定頁建立管理者帳號
- 管理者可在後台建立聚會並控制是否開放報名
- 統計報名人數、葷食數量、素食數量
- 可選擇是否用餐，用餐才填寫葷食與素食數量
- 匯出 CSV
- 手機優先 RWD，適合 LINE 瀏覽器開啟

## 使用情境

- 同工建立聚會並開放報名
- 參加者填寫邀請人、會所、區、受邀人與用餐資訊
- 參加者可回到報名頁查詢既有報名並修改
- 管理者查看報名清單、統計人數與餐點數量
- 聚會結束或報名截止後，管理者可關閉報名

## 管理與資料

- 後台僅開放授權管理者使用。
- 報名資料儲存在 Supabase，並透過資料庫權限控管公開與管理行為。
